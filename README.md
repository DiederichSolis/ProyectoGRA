#Objetivo

<P>
Crear un diorama de minecraft usando o implementando los temas visto en clase....
</P>

#Running the Project
<p>
1.) clonar repositorio
</p>
```
gti clone https://github.com/DiederichSolis/ProyectoGRA.git
```
<p>
2.) Pasarse a la carpeta minecraft
</p>
```
cd minecraft
```
<p>
3.) Realizar el build
</p>
```
cargo build
```

4.) Correrlo
</p>
```
cargo run --release
```
